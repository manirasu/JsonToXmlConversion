JSON to XML Conversion
This project converts the json to xml based on this given specification. We need to pass mandatory json file path as first argument and optional output xml file path as second parameter. 

Build :

mvn package

output:
It will generate <Jsontoxml.jar> jar file in System.getProperty("user.dir")/target directory.

Run:
Java -jar < jar file>  <input json path>  <output xml path>
